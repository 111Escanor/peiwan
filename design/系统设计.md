## 技术选型
- 后端：Spring Boot + MyBatis Plus + JWT
- 前端：Vue + Element UI + Axios
- 数据库：MySQL 8.0

## 模块划分
### 用户模块
- 实现注册、登录、个人信息管理
- 使用 BCrypt 加密密码，JWT 生成 token

### 订单模块
- 状态机：pending -> paid -> in_service -> completed
- 使用乐观锁防止并发接单

### 支付模块
- 模拟支付接口，记录交易流水

### 陪玩师模块
- 认证状态管理，技能标签
